__깃허브 업데이트 다운__
```
cd 프로젝트 경로
git pull
파일탐색기 -> 프로젝트 경로-> src 안의 폴더/파일을 이클립스 같은 경로에 드래그앤드롭 -> overwrite all
```

__db연결__
```
아래창의 data source explorer 클릭 -> database connections -> 만든 db 더블클릭
```

__sql 위치__
```
src/main/WebContent/resource/sql/sanhak_sql.sql
```

__sql 세팅__
```
윗부분 Connection profile type에 Mysql_5.1, Name에 본인 DB명, Database에 본인 DB 세팅
```

__sql 실행__
```
alt+x -> 여러 줄 실행, alt+s -> 한 줄 실행
```
